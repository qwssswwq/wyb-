# 计算机基础与学习路线

## 1. OSSU Computer Science

- 原项目：[ossu/computer-science](https://github.com/ossu/computer-science)
- 用途：按照大学计算机科学培养体系组织的免费自学课程。
- 适合：希望系统补齐编程、算法、数学、系统和理论基础的学习者。
- 建议：把它作为长期课程表，不建议一次性下载所有课程材料。

## 2. Free Programming Books

- 原项目：[EbookFoundation/free-programming-books](https://github.com/EbookFoundation/free-programming-books)
- 用途：多语言免费编程书籍、课程和学习资源索引。
- 适合：查找特定语言、框架和技术方向的学习资料。
- 注意：每份外部资源可能有不同版权与使用条件。

## 3. Developer Roadmap

- 原项目：[nilbuild/developer-roadmap](https://github.com/nilbuild/developer-roadmap)
- 用途：前端、后端、DevOps、AI等开发方向的学习路线图。
- 适合：确定学习顺序、检查技能缺口。
- 注意：路线图是参考，不需要机械完成所有节点。

## 4. Project Based Learning

- 原项目：[practical-tutorials/project-based-learning](https://github.com/practical-tutorials/project-based-learning)
- 用途：按编程语言分类的项目实战教程。
- 适合：希望通过完成真实项目学习编程的人。
- 建议：每个阶段选择一个项目完成，不要只收藏不实践。

## 5. Coding Interview University

- 原项目：[jwasham/coding-interview-university](https://github.com/jwasham/coding-interview-university)
- 用途：数据结构、算法与计算机面试学习计划。
- 适合：准备软件工程师面试或系统复习基础知识。
- 注意：部分内容面向面试，不等于完整的软件工程实践。

## 6. System Design Primer

- 原项目：[donnemartin/system-design-primer](https://github.com/donnemartin/system-design-primer)
- 用途：系统设计概念、架构案例与面试题。
- 适合：后端、架构和高并发系统学习。
- 注意：示例架构需要结合真实业务规模判断，不能直接照搬。
