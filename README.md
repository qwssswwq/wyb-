# 编程优秀资源导航

这是一个面向中文开发者的优质编程仓库索引，集中整理值得学习和参考的开源项目。

> 本仓库只保存项目介绍和原始链接，不复制第三方源码。项目版权和许可证归原作者所有；使用、修改或分发代码前，请阅读对应仓库的 LICENSE。

## 分类目录

- [计算机基础与学习路线](catalog/learning-roadmaps.md)
- [开发工具与工程资源](catalog/developer-resources.md)
- [AI 与大模型编程](catalog/ai-development.md)

## 第一批收录标准

- 原始仓库真实存在且仍可访问
- 内容具有长期学习或工程参考价值
- 优先选择维护成熟的课程、路线图、Awesome清单和官方示例
- 不直接搬运代码，避免版权、更新和安全风险
- 收录不等于安全背书，依赖或运行代码前仍需自行审查

## 如何使用

1. 根据分类打开目标项目。
2. 阅读项目 README、LICENSE、最近提交和 Issues。
3. 只对确实需要修改的项目使用 Fork。
4. 项目依赖优先使用官方包管理器，不要手工复制整个源码。
5. 对长期关注的项目使用 GitHub Star 与 Lists。

## 当前规模

第一批共收录 12 个优质仓库，后续可以继续按语言、框架和业务方向扩充。

## 更新原则

新增资源时建议同时记录：

- 项目名称与原始链接
- 适用人群和主要用途
- 推荐理由
- 使用前注意事项
- 许可证与安全检查状态
